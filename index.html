About Me

My name is Thuc Nguyen, but I also go by Ennoia. I graduated from UCSD in 2017 with a Human Biology major. During the years as an undergraduate, I grew a strong interest in Computer Science and started learning how to program offhand. I looked into C++ as my first programming language to gain some fundamental understanding of how to code. Thereafter, I sought to learn a multitude of languages and coded some mobile applications along the way. The only computer science classes I took during my undergrad years included an Introduction to Java class and a Basic Data Structures class. However, I am now currently enrolled in a post-baccalaureate program for Web and Mobile Application Development at SDSU. Despite my lack of a formal education in this field, I am motivated to continue to pursue C.S. and complete as many projects as I can. This is my life's work.
