Variables

Variables: declared with var keyword. Use for storage, can be reused and set to different primary types - String, Number, Boolean, or some Object.

Global variables: declared outside the function, alive as long as page is laoded.

Local variables: accessible within the scope of the function.

Note: a variable defined within scope of a local function without the var keyword becomes global

Some JS Functions

document.write("Hello World"); - writes out the text to the document

alert(message); - alerts user with popup and the message

confirm(); - asks user if want to confirm, returns true or false value

Functions

Functions: set of statements to perform a task, can't put numbers in beginning of function name

Arguments are passed in as values to parameters of functions, they are copied to the parameters.

Ex.

            function addTogether(num, str){
                var add = num + str;
                alert(add);
            }
            addTogether(100, " things I like");

Can return from function and store it in a variable

            function add(x, y){
                result = x + y;
                return result;
            }
            var sum = add(2, 3);

Objects

Properties: variables inside an object, use dot syntax (.) to access properties. Properties describe the object, actions describe the behavior

Methods: actions of the object that describes its behavior

new keyword: creates copy of existing object or create an instance of it

delete keywrod: deletes property

            var otter = {
                color: "grey",
                class: "mammalia"
                swim: function(){
                    document.write("the otter swam");
                }
            }

Can change values of properties in object and its data type. Can add new properties as long as the property has a different name.

otter.numLimbs = 4; - added new property numLimbs with value 4

delete numLimbs - deleted property numLimbs

Document Object Model (DOM)

Every html element has a corresponding object, allows us to use JS to access parts of the web page. Acts as the interface to get to the HTML. DOM represents entire web page or document.

The browser loads the page, then the DOM is created. Objects are stored in the DOM.

Nodes: Element node, Attribute node, and Text node

Some of the things you can do after selecting an element to target is that you can change the CSS style, the text, the image, perform validation, create new pages, and change internals.

Ways to Reference HTML elements

  1. getElementById();
  2. getElementsByClassName()
  3. getElementsByTagName()
            var element = document.getElementById("idOfElement");
            var element = document.getElementsByClassName("className");
            var element = document.getElementsByTag("tag");

For both getting elements by class and by tag name, a node list is returned. List must be iterated over of have its indices accessed to get the actual element.

Changing Style of elements

            element.style.property = "";
            element.style.color = "blue";
            element.style.backgroundColor = "red";
            element.style.fontStyle = "italic";
            and so on...

JS doesn't recognize hyphen within properties of CSS, so it must be camelcase. Exception is with padding.

innerHTML

innerHTML is the HTML content of the element grabbed, gets the text child node of the element.

var content = element.innerHTML; - reads the content

element.innerHTML = text; - sets the content

Image

Can use src attribute of image tag to change the image

imageElement.src = "pathToNewImage";

Events

EventListeners can be attached to the HTML element and the EventHandler is the piece of functionality that will run when that event occurs. We can choose the Event for the object to listen to. The "on" event that's usually defined in HTML as an attribute is left out in JS.

            element.addEventListener(event, functionName);
            functionName() {...}

List of Events

  • onclick
  • onmouseover
  • onmouseout

Creating and Removing elements

            Creating a new element:
            var newElement = document.createElement("htmlElement");
            someDiv.appendChild(newElement);
            var text = document.createTextNode("someText");
            newElement.appendChild(text);

            Removing an element:
            var parentNode = childElement.parentNode();
            parentNode.removeChild(childElement);

Creating Attribute Node

            var newAttribute = document.createAttribute("attribute");
            newAttribute.value = "attributeValue";
            element.setAttributeNode(newAttribute);

Traversing the DOM

Use the parent-child relationship to move around and target an element. Can go from parent to child, child to parent, child to siblings, and so on.

Note that when returning a node list, this includes whitespace nodes.

            Parent to Child:
            var childNodes = parent.childNodes[];   // Returns list of all child nodes
            var firstElement = parent.firstElementChild;    // Give first child element
            var lastElement = parent.lastElementChild;      // Give last child element

            Child to Parent:
            var parent = child.parentNode;      // if only 1 parent
            var parent = child.parentElement;   // gives parent element

            Child to Siblings:
            var sibling = element.nextElementSibling    // gives element after
            var sibling = element.previousElementSibling    // gives element before

Forms

For forms, value property of the html element object returns the value of the input. Can do value.length to grab the length of the input value.

Classes

Using the Local Storage of the Browser

                if(!localStorage){

                    return;

                var theKey = document.getElementById("key").value;

                localStorage.setItem(theKey, theValue);

                if (!localStorage){

                }
                var theKey = document.getElementById("findKey").value;
                var theValue = localStorage.getItem(theKey);
                var handle = document.getElementById("results");
                handle.innerHTML = theValue;
