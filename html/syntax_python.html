While Loops

Can include multiple conditions

    while j < 5:

    # Example of multiple conditions

Lambda

Lambda: a small anonymous function. Can take any number of arguments but can only have one expression. Usually used as anonymous function inside another function.


    lambda arguments : expression
    


    # The following adds 10 to the number passed in as an argument
    # and prints the result
    x = lambda a : a + 10
    print(x(5))    # prints 15

    x = lambda a, b : a * b
    print(x(5, 6))    # prints 30

    # Using lambdas in another function
    def my_func(n):
        return lambda a : a * n

    # double_func is now a lambda that takes a number as an argument
    # and multiplies it by 2 (since the lambda was returned)
    double_func = my_func(2)

    print(double_func(11))    # prints 22

Classes and Objects

  • Capitalize the class name.
  • Keyword self is similar to this in Java.
  • Functions within a class, or methods, must have an additional argument called self. It is a reference to the class itself and allows access to variables belonging to the class.
  • If an attribute or property doesn't exist and a method is ran which expects that attribute, an error will be thrown. To avoid this, it's better to use a constructor to assign values to object properties. The keyword for this constructor is __init__
  • The convention is to name the parameters to the constructor the same as the attributes that you want to set.
  • Once a custom constructor is defined in the class, the default constructor won't work anymore.
  • Remember to pass self as the first argument to the constructor's parameters.


    class Robot:
        x = 5    # a property of the class

        def introduce_self(self):
            print("My name is " + self.name)

    # Using default constructor for Robot class
    r1 = Robot()

    # Set attributes
    r1.name = "Tom"
    r1.color = "red"
    r1.weight = 30
    r1.introduce_self()    # prints "My name is Tom"

    r2 = Robot()
    r2.name = "Jerry"
    r2.color = "blue"
    r2.weight = 40
    r2.introduce_self()    # prints "My name is Jerry"

    # Delete object properties
    del r1.weight

    # Delete entire objects
    del r1

    # Custom constructor
    class Robot:
        def __init__(self, name, color, weight):
            self.name = name
            self.color = color
            self.weight = weight

    r1 = Robot("Tom", "red", 30)
    r2 = Robot("Jerry", "blue", 40)

Iterators

Iterator: an object that contains a countable number of values, and can be traversed upon through all the values.

An iterator is an object that implements the iterator protocol, so it must implement the methods -

  • __iter__()
  • __next__()

Collections and strings are iterable objects. Use the iter() method to get an iterator.



    my_tuple = ("apple", "banana", "cherry")
    my_iter = iter(my_tuple)
    print(next(my_iter))    # prints "apple"
    print(next(my_iter))    # prints "banana"
    print(next(my_iter))    # prints "cherry"

Creating an Iterator

__iter__(): must always return the iterator object itself. Can initialize values here too.

__next__(): must return the next item in the sequence.

StopIteration: statement that terminates the iteration, raises the error.



    class MyNumbers:
        def __iter__(self):
            self.a = 1
            return self

        def __next__(self):
            if self.a <= 20:
                x = self.a
                self.a += 1
                return x
            else:
                raise StopIteration

    my_class = MyNumbers()
    my_iter = iter(my_class)

Try, Except

A try block tests a block of code for errors, an except block handles the error, and a finally block lets you execute the code regardless of the try and catch blocks (useful for closing objects and cleaning up resources). Can use else to define block of code if no errors were raised.



    try:
        print(x)
    except NameError:
        print("Variable x is not defined")
    except:
        print("Something else went wrong")
    else:
        print("Nothing went wrong")
    finally:
        print("The 'try except' is finished")

Modules

Module: a code library, a file containing a set of functions that you want to include in your application.

  • Create one by saving code in a file with the .py extension
  • Use module by using the import statement. Use from keyword to import only parts of a module. If we do import using from, don't use module name to refer to elements in the module since it is not needed.
  • Can use alias when importing a module by using the as keyword.
  • Can use built-in modules such as platform
  • Use dir() function to list the function names or variable names in a module.


    # Example of importing
    import mymodule

    # To use function from a module: module_name.function_name
    mymodule.greeting("Jonathan")

    # Accessing a variable from a module
    module_name.variable

    # Using an alias when importing
    import mymodule as mx

Termination

To exit a script, the following can be done which raises a SystemExit exception. This will also terminate all associated python scripts.



    import sys

    sys.exit()

Can also use quit() to terminate. Don't need to import and the function will only terminate the script that spawned it.

global keyword

global is used to access and modify variables outside of a function, or within the module.



    some_variable = 0

    def change_value():
        global some_variable

        some_variable = 5

    print(some_variable)    # prints 0
    change_value()
    print(some_variable)    # prints 5
