JavaScript Syntax

Javascript

JavaScript: adds programming to web pages. Places code into page and browser executes it. A loosely typed language, don't need to explicitly declare data types. All about tying function to an event.


Place in <script> element tag within <head> and either allow page to load before calling JS functions or use JQuery's document.ready() function.


    <html>
        <head>
            <script src="javascript_file.js"><⁄script>
        <⁄head>
        <body>
        <⁄body>
    <⁄html>

JavaScript Types

The following table shows primitive types and their primitive values all of which are immutable except object types.


Type True False

Difference between undefined and null

undefined: a variable has been declared but not yet assigned a value. It is a type itself.

null: an assignment value representing no value. It is an object.

Variables

Variables: declared with var keyword. Use for storage, can be reused and set to different primary types.


Global variables: declared outside the function, alive as long as page is loaded.


Local variables: accessible within the scope of the function.

  • A variable defined within scope of a local function without the var keyword becomes global.


    var day;    // Declare variable without assigning value
    var value = 100;    // Declare variable and assign value
    var number = 1;    // Initialize with numeric literal
    var text = "hello";    // Initialize with a String
    var status = true;    // Initialize with a Boolean value
    callingFunction(parameter);    // Call function and pass in parameters

Loops

Aside from the while loop and the standard for loop, there is also the for...in and for...of loops

  • for...in iterates over enumerable properties of an object. It shouldn't be used to iterate over an array where index order is important because there is no guarantee that the indexes will be in order.
  • for...of iterates over iterable objects. It should be used to iterate over an array, returns the value of what was iterated over.


    // The while loop
    while (conditions) {
        statements;
    }

    // Standard for loop
    for (counter; condition; incrementor) {
        statements;
    }

    // The for...in loop
    for (variable in object) {
        statements;
    }

    // The for...of loop
    for (variable of iterable) {
        statements;
    }

Functions

Functions: set of statements to perform a task, can't put numbers in beginning of function name

  • The () operator invokes the function. Accessing the function without () gives the function definition and not the result.
  • Functions can be used as the value for a variable.

Arguments: passed in as values to parameters of functions, they are copied to the parameters. Can be accessed using arguments keyword and indexing.

  • Accessing arguments - arguments objects have entry for each argument passed to function, but doesn't have array properties aside from length.

Function Syntax


    function name (parameter1, parameter2, parameter3) {
        statements;
    }

Examples of Functions


    function addTogether(num, str){
        var add = num + str;
        alert(add);
    }
    addTogether(100, " things I like");

    // Can return from function and store it in a variable
    function add(x, y){
        result = x + y;
        return result;
    }
    var sum = add(2, 3);

    // Accessing Arguments
    function someFunction(){
        alert(arguments[0]);
    }

    element.addEventListener("event", function() {

    document.write("string");    // writes out the text to the document

Enums

Enum: a type that restricts variables to one value from a predefined set of constants.

  • Javascript doesn't really have enums. They are more like objects with properties that can be accessed and added to, even when those properties are not yet declared.
  • Can also have properties that are enums.


    // Example of creating an enum
    var SomeEnum = {
        PROPERTY_1: value,
        PROPERTY_2: value,
        PROPERTY_3: value
    };
    // Setting a variable to an enum value
    var currentValue = SomeEnum.PROPERTY_1;

Arrays

Arrays won't give an array IndexOutOfBoundsException.


    var array = [value1, value2, value3];    // initialize array with values
    var getValue = array[index];    // get value at position/index
    array[index] = newValue;    // set value at position/index
    var multiDimArray = [ [x, x, x], [y, y, y] ];    // 2D array

Objects

Properties: variables inside an object, use dot syntax (.) to access properties. Properties describe the object, actions describe the behavior.

Methods: actions of the object that describe its behavior.

new keyword: creates copy of existing object or create an instance of it.

delete keyword: deletes property.


    var otter = {
        color: "grey",
        animalClass: "mammalia",
        swim: function(){
            document.write("The otter swam.");
        }
    }

Can change values of properties in object and its data type. Can add new properties as long as the property has a different name.


    otter.numLimbs = 4;    // added new property numLimbs with value 4
    delete otter.numLimbs;    // deleted property numLimbs

Example of using a Random Number Generator

  • Math.random() returns a random number between 0(inclusive) and 1(exclusive)
  • To get the random number to expand to some specified range, multiply it by that number.


    // Returns a random number from 1-10
    Math.floor((Math.random() * 10) + 1);

Number Object

toFixed(): formats a number using fixed-point notation. Returns a string representation of the number object.

  • [digits] is the # of digits to appear after the decimal point. Can only be between 0 and 20.

    numObj.toFixed([digits]);

Date Object

For the constructor for the Date object, whatever isn't filled in will be taken as 0. Months are zero-based.


    var dateObj = new Date();    // Returns current date and time
    var dateObj = new Date(year, month, day, hrs, min, sec, millis);

parseInt() and parseFloat()

parseInt(string, radix): parses String and returns integer of specified radix, the base

  • Specify 10 for radix for decimal numeral system.

parseFloat(value): parses argument and returns a floating point number. Returns NaN if value can't be parsed.

Windows object

The document object that comprises the entire webpage. It is the browser window itself. Usually use this object for page redirects.


    window.location="url"

Setting a timeout on the window object

setTimeout(function, ms, param1, param2...): calls a function after a specified number of milliseconds

  • Can use this to redirect user to a page after waiting for an amount of time.

    window.setTimeout(function() {
        window.location.href="...";
    }, ms);

Document Object Model (DOM)

DOM: represents the entire webpage or document. Every html element has a corresponding object, can use JS to access parts of the web page. Acts as the interface to get to the HTML.

  • The browser loads the page, then the DOM is created. Objects are stored in the DOM.
  • After selecting an element to target, can change the CSS style, the text, the image, perform validation, create new pages, and change internals.

Nodes: Element node, Attribute node, and Text node


Loading of the DOM

DomContentLoaded event fires when the DOM finishes loading and the markup is parsed.

Load event fires when page is fully loaded, including downloading images and external resources.



    // Can add the following event listeners
    document.addEventListener("DocumentContentLoaded", domLoaded, false);
    window.addEventListener("load", pageLoaded, false);
    function domLoaded() {
        statements;
    }
    function pageLoaded() {
        statements;
    }

Ways to Reference HTML elements

  1. getElementById();
  2. getElementsByClassName();
  3. getElementsByTagName();

    var element = document.getElementById("idOfElement");
    var element = document.getElementsByClassName("className");
    var element = document.getElementsByTag("tag");

For both getting elements by class and by tag name, a node list is returned. List must be iterated over of have its indices accessed to get the actual element.

Changing Style of elements


    element.style.property = "";
    element.style.color = "blue";
    element.style.backgroundColor = "red";
    element.style.fontStyle = "italic";
    etc.

JS doesn't recognize hyphen within properties of CSS, so it must be camelcase. Exception is with padding.

innerHTML

innerHTML is the HTML content of the element grabbed, gets the text child node of the element.


    var content = element.innerHTML;    // reads the content
    element.innerHTML = text;    // sets the content

Image

Use src attribute of image tag to change the image


    imageElement.src = "pathToNewImage";

Events

EventListeners can be attached or removed to or from the HTML element and the EventHandler is the piece of functionality that will run when that event occurs. Can choose the Event for the object to listen to. The "on" event that's usually defined in HTML as an attribute is left out in JS.

  • Can add multiple Event Listeners to an element and event handlers of the same type.
  • Can add event listeners to DOM objects too like the window object.


    // Attaching an event listener
    element.addEventListener("event", functionName);

    // Removing an event listener
    element.removeEventListener("event", functionName);

List of Events

  • onclick
  • onmouseover
  • onmouseout

Event Object

event.target: the target property can be the element that registered for the event or a descendant of it. It is a reference to the object in which the event was triggered.

  • We can compare event.target to this to determine if the event is being handled due to event bubbling.

Creating and Removing elements


    // Creating a new element:
    var newElement = document.createElement("htmlElement");
    someDiv.appendChild(newElement);
    var text = document.createTextNode("someText");
    newElement.appendChild(text);

    // Removing an element:
    var parentNode = childElement.parentNode();
    parentNode.removeChild(childElement);

Creating Attribute Node


    var newAttribute = document.createAttribute("attribute");
    newAttribute.value = "attributeValue";
    element.setAttributeNode(newAttribute);

Traversing the DOM

Use the parent-child relationship to move around and target an element. Can go from parent to child, child to parent, child to siblings, etc.

  • When returning a node list, the list will include whitespace nodes.


    // Parent to Child:
    var childNodes = parent.childNodes[];    // Returns list of all child nodes
    var firstElement = parent.firstElementChild;    // Give first child element
    var lastElement = parent.lastElementChild;    // Give last child element

    // Child to Parent:
    var parent = child.parentNode;    // if only 1 parent
    var parent = child.parentElement;    // gives parent element

    // Child to Siblings:
    var sibling = element.nextElementSibling    // gives element after
    var sibling = element.previousElementSibling    // gives element before

Forms and Validation

For forms, value property of the html element object returns the value of the input. Can do value.length to grab the length of the input value.


Input Type Radio - checked property can be used to set the checked property or to get the boolean value of whether or not the radio button was checked.


    // Returns true or false depending on if radio is checked
    var value = radioObject.checked;

    // Set the checked property
    radioObject.checked = true | false;

Javascript Regular Expressions

Regular Expression(Regex): sequence of characters that defines a search pattern

  • /^pattern$/modifiers
  • To create a Regular Expression object, use the RegExp constructor

Visit these sites for a more detailed explanation: GeeksforGeeks and MDN web docs



    var regexObj = new RegExp(pattern, flags);

Regex Modifiers
i Matching will be case-insensitive.
g Matching will be global.

Regex Patterns
[abc] Find any character that matches the ones in the bracket.
[0-9] Find any digits matching the ones within the brackets.

Regex Metacharacters
\d Finds a digit.
\s Finds a whitespace character.

Regex Quantifiers
n+ At least one n.
n* Zero or more occurrences of n.
n? Zero or one occurrences of n.
{...} Repeat preceding character for as many times as the value inside the bracket.

Animations and Transitions

Animations

For animations to work, the following conditions must be met:

  • Animations should be relative to a container.
  • The container should have position relative and the element to animate should have position absolute.

The element's style is changed over time by using a timer. Use setInterval(function, time) and clearInterval(id).


    var id = setInterval(someFunction, time);
    function someFunction() {
        if (condition_for_completion) {
            clearInterval(id);
        } else {
            // change element style
        }
    }

Transitions

Set the value for the transition property.


    element.style.transition = [property] [duration] [transition-timing-function] [delay];

    // Example
    element.style.transition = "width 1s ease 0s";

Javascript Transition Events

The transitionend event is widely supported and fires when the transition of an element is completed.



    element.addEventListener("transitionend", function() {
        alert("transition ended");
    });
