api.jquery.com contains the official documentation for the JQuery language and syntax. This page selects and summarizes some of the topics, properties, and methods of JQuery.

JQuery

JQuery: an extension of JavaScript, a JS framework and open-source library. Lots of features that are more high-level and allows cross browser compatibility.

  • Uses CSS Selector methodology which makes selecting DOM elements easier.
  • It's all about learning the selectors and methods/properties to use.

either 1.x or 2.x, the deveopment and production version. Production version is compressed.

JQuery Syntax


    // Single Selector
    $("selector").method();

    // Multiple Selectors
    $("selector1, selector2, selector3").method();

    // Descendant Selector
    $("parentSelector childSelector").method();

$ represents JQuery library. Selectors can be any id, class, tag, type, descendant, and any other CSS selectors.

Specifying JQuery in HTML code

Content Delivery Network(CDN): system of servers that delivers pages/web content to user based on geographical proximity.

  1. Place the script tag with the src attribute linking to the JQuery library in the head section. The CDN can be used here if you decide not to download the library.
  2. Below that, place the script tag with the src attribute linking to the javascript file.


    <!-- HTML -->
    <script src="jquery-1.11.3.js"><⁄script>
    <script src="javascriptFile.js"><⁄script>

Starting Template

The following is in an external javascript file.


    $(document).ready(function () {
        $("selector").method();
    });

ready function waits until webpage is downlaoded before any jquery code is executed. Usually do your setup here like initializing variables or grabbing html elements as objects and setting EventListeners to them.

JQuery Events

The following are a list of some of the events that can be attached to an object. The shorthand way is displayed first. The function can be anonymous and defined as a parameter or a function name without the invoker can be set as the parameter.


    // The following is an anonymous call to a function
    $("selector").on("event", function() {
        statements;
    });

    // The following is using the function name, usually for reusability
    $("selector").on("event", functionName);
Events Code
Click: event is fired when user clicks on the html element. 

    $("selector").click(function() {
        statements;
    });

    $("selector").on("click", function() {
        statements;
    })
Mouseover: event is fired when user mouses over html element. 

    $("selector").mouseover(function() {
        statements;
    });
Load: event that is sent to an element when it and all sub-elements have been completely loaded. Sent to element associated with a URL: images, scripts, frames, iframes, and window object. 

    $("selector").load(function() {
        statements;
    });

JQuery Selectors

JQuery selectors are just like the CSS selectors since the library uses the CSS methodology.

For the following, add the selectors and the pseudo class or element in place of "selector"


    $("selector:pesudoclass").method();
Selector Definition
:first and :last Selects first and last element, respectively.
:empty Selects elements or filtered elements that are empty.
:even and :odd Selects all even numbered or odd numbered elements or filtered elements, respectively.
:nth-child(n) Selects child element by index n, it not zero-based.
:not() Applies to all elements or filtered elements that aren't within the not parameter.
:contains() Selects all elements or filtered elements containing the text.
:has() Selects elements that have specified child element.

JQuery Methods

Methods can be chained using dot syntax (.)


    Ex.
    $("selector").css("property", "value").css("property", "value")...
Method Code
CSS method: sets values to the properties of the selector 

    // For single rule
    $("selector").css("property", "value(s)");

    // For multiple rules
    $("selector").css({
        "property": "value(s)",
        "property": "value(s)"
    });
HTML method: replaces actual html element along with any applicable text. 

    $("selector").html("<tag>Content<tag>");
Text Method: like html method but only replaces the text. 

    $("selector").text("Content");
Each method and this keyword: each lets looping through elements of selector, a DOM looping construct. this keyword is the current element of the loop. 

    $("selector").each(function() {
        $(this).method();
    });
Add, Remove, and Toggle Classes: adds or removes class attribute to elements, or toggle the class on and off 

    $("selector").addClass("className");
    $("selector").removeClass("className");
    $("selector").toggleClass("className")
Append, Prepend, After, and Before: append of prepend selected element with the new element along with its content within the selector (after or before its first child element). After and Before will place the element outside of the selector. 

    // Append or Prepend
    $("selector").append("<tag>Content<⁄tag>");
    $("selector").prepend("<tag>content<⁄tag>");

    // After or Before
    $("selector").after("<tag>Content<⁄tag>");
    $("selector").before("<tag>Content<⁄tag>");
Replace and Remove: replaces selector with new element or removes if completely. 

    // Replace
    $("selector").replace("<tag>Content<⁄tag>");

    // Remove
    $("selector").remove();
Attribute: sets or removes the attribute to and from the selector. Examples include src= for img tag, pattern= for input, etc. 

    // Add Attribute
    $("selector").attr("attribute", value"");

    // Remove Attribute
    $("selector").removeAttr("attribute");
Hover: methods takes two functions, one for hover on and the other for hover off. 

    $("selector").hover(function() {
        statements;
    }, function() {
        statements;
    });
Fade In, Fade Out, Toggling Fade, and Fade To: fades in or fades out the element. Toggle fading on and off, or fade to a specific opacity 

    // Fade In, Fade Out, Toggle
    // value = slow, fast, or specific milliseconds(ms)
    $("selector").fadeIn(value);
    $("selector").fadeOut(value);
    $("selector").fadeToggle(value);

    // Fade To
    // value1 = slow, fast, or milliseconds (ms)
    // value2 = opacity (0.0-1.0)
    $("selector").fadeTo(value1, value2);
Slide Effects and Toggling 

    $("selector").slideUp();
    $("selector").slideDown();
    $("selector").slideToggle();
Show and Hide: usually for images and menus 

    // Show the selected element
    $("selector").show();

    // Hide the selected element
    $("selector").hide();
