JQuery

JQuery: an extension of JavaScript, a JS framework and open-source library. Lots of features that are more high-level and allows cross-browser compatibility.

  • Uses CSS Selector methodology which makes selecting DOM elements easier.
  • It's all about learning the selectors and methods/properties to use.

Download JQuery either 1.x or 2.x, the deveopment and production version. Production version is compressed.

JQuery Syntax

$ represents JQuery library. Selectors can be any id, class, tag, type, descendant, and any other CSS selectors.



// Single Selector
$("selector").method();

// Multiple Selectors
$("selector1, selector2, selector3").method();

// Descendant Selector
$("parentSelector childSelector").method();

Specifying JQuery in HTML code

Content Delivery Network(CDN): system of servers that delivers pages/web content to user based on geographical proximity.

  1. Place the script tag with the src attribute linking to the JQuery library in the head section. The CDN can be used here if you decide not to download the library.
  2. Below that, place the script tag with the src attribute linking to the javascript file.


<!-- HTML -->
<script src="jquery-1.11.3.js"><⁄script>
<script src="javascriptFile.js"><⁄script>

Starting Template

ready() function waits until webpage is downloaded before any jquery code is executed. Usually do the setup here such as initializing variables or grabbing HTML elements as objects and setting Event Listeners to them.


The following is in an external javascript file.


$(document).ready(function () {
    $("selector").method();
});

Difference between onload() and $(document).ready()

onload(): won't finish until everything on the page has loaded, will wait until external resources are downlaoded.

$(document).ready(): executes right when DOM is built and in memory.

JQuery object vs DOM object

Using JQuery selectors returns all the elements in the document that match in a collection. DOM elements can be wrapped in a JQuery object to access methods that are more concise and straightforward. JQuery objects can access different methods than Javascript objects, so these must be distinguished using the wrapper $("selector").


In order to convert a JQuery object to a Javascript object, brackets can be used. Can use [0] because a JQuery object is always a collection, and you are retrieving the first object that matches by getting the 0th index.


// index is an integer
var domElement = $("selector")[index];

// Example
var formData = new FormData($('form')[0]);

JQuery Selectors

JQuery selectors are just like the CSS selectors since the library uses the CSS methodology.

For the following, the pseudo class can be added after "selector".


$("selector:pesudoclass").method();

Pseudo Element Definition
:first and :last Selects first and last element, respectively.
:empty Selects elements or filtered elements that are empty.
:even and :odd Selects all even numbered or odd numbered elements or filtered elements, respectively.
:nth-child(n) Selects child element by index n, it not zero-based.
:not() Applies to all elements or filtered elements that aren't within the not parameter.
:contains() Selects all elements or filtered elements containing the text.
:has() Selects elements that have specified child element.

Differences between this and $(this)

The this keyword is the specific DOM element associated with the event that was triggered.


$(this) still refers to the currently selected object but because it is wrapped in a JQuery object, it has access to JQuery methods. The wrapper only adds the JQuery library to the reference.

JQuery Methods

Implicit Iteration: applies the rule while going through the objects in the collection.


Methods can be chained using dot syntax (.)


Ex.
$("selector").css("property", "value").css("property", "value")...

Method Code
css(): sets values to the properties of the selector 

// For single rule
$("selector").css("property", "value(s)");

// For multiple rules
$("selector").css({
    "property": "value(s)",
    "property": "value(s)"
});
html(): replaces html element along with any applicable text. Can use this method to return the HTML contents of the element. 

$("selector").html("<tag>Content<tag>");
text(): like html method but only replaces the text. 

$("selector").text("Content");
each() and this: each lets looping through elements of selector, a DOM looping construct. this keyword is the current element of the loop. 

$("selector").each(function() {
    $(this).method();
});
eq(): returns the specific JQuery object at the specified index in the collection of JQuery objects. 

// index is an integer
var jqueryObj = $("selector").eq(index);
get(): similar to eq() but returns the DOM element itself. 

// index is an integer
var domElement = $("selector").get(index);
val(): gets the current value of the element or sets the value of every matched element. Usually used to get value of form elements. 

// Get the value
var theValue = $("selector").val();

// Set the value
$("selector").val(value);
children(selector): matches the children of the element but only a single level down the DOM tree. 

$("selector").children("selector");
find(selector): matches the children of the element(s) but goes through its descendants. 

$("selector").find("selector");
closest(selector): get the first element that matches the selector by testing the element itself and then traversing up the tree to the ancestors. 

$("selector").closest("selector");
parents(): also traverses up the tree to find a match but begins with the parent element and continues to find all the ancestors that matches, instead of stopping when finding the first match. 

$("selector").parents("selector");
Add, Remove, and Toggle Classes: adds or removes class attribute to elements, or toggle the class on and off 

$("selector").addClass("className");
$("selector").removeClass("className");
$("selector").toggleClass("className")
Append, Prepend, After, and Before: append of prepend selected element with the new element along with its content within the selector (after or before its first child element). After and Before will place the element outside of the selector. 

// Append or Prepend
$("selector").append("<tag>Content<⁄tag>");
$("selector").prepend("<tag>content<⁄tag>");

// After or Before
$("selector").after("<tag>Content<⁄tag>");
$("selector").before("<tag>Content<⁄tag>");
Replace and Remove: replaces selector with new element or removes if completely. 

// Replace
$("selector").replace("<tag>Content<⁄tag>");

// Remove
$("selector").remove();
Attribute: sets or removes the attribute to and from the selector. Examples include src= for img tag, pattern= for input, etc. 

// Add Attribute
$("selector").attr("attribute", value"");

// Remove Attribute
$("selector").removeAttr("attribute");
Hover: methods takes two functions, one for hover on and the other for hover off. 

$("selector").hover(function() {
    statements;
}, function() {
    statements;
});
Fade In, Fade Out, Toggling Fade, and Fade To: fades in or fades out the element. Toggle fading on and off, or fade to a specific opacity 

// Fade In, Fade Out, Toggle
// value = slow, fast, or specific milliseconds(ms)
$("selector").fadeIn(value);
$("selector").fadeOut(value);
$("selector").fadeToggle(value);

// Fade To
// value1 = slow, fast, or milliseconds (ms)
// value2 = opacity (0.0-1.0)
$("selector").fadeTo(value1, value2);
Slide Effects and Toggling 

$("selector").slideUp();
$("selector").slideDown();
$("selector").slideToggle();
Show and Hide: usually for images and menus 

// Show the selected element
$("selector").show();

// Hide the selected element
$("selector").hide();

JQuery Events

The following is a list of some of the events that can be attached to an object. The shorthand way is displayed first. The function can be anonymous and defined as a parameter or a function name without the invoker can be set as the parameter.


// The following is an anonymous call to a function
$("selector").on("event", function() {
    statements;
});

// The following is using the function name, usually for reusability
$("selector").on("event", functionName);

Events Code
Click: event is fired when user clicks on the html element. 

$("selector").click(function() {
    statements;
});

$("selector").on("click", function() {
    statements;
});
Mouseover: event is fired when user mouses over html element. 

$("selector").mouseover(function() {
    statements;
});
Load: event that is sent to an element when it and all sub-elements have been completely loaded. Sent to element associated with a URL: images, scripts, frames, iframes, and window object. 

$("selector").load(function() {
    statements;
});

Event Object

event.preventDefault(): prevents the default action of the event from triggering.

  • For example, you could use this on clicked anchors so that it won't take the user to the URL.

Event Delegation

Visit this website for more detailed information


Event Delegation: attaching an event listener to a parent container, which will trigger that event through all of its children.

  • This allows the possibility of adding event listeners to elements that doesn't exist yet, or that will be dynamically added later on.
  • Note that event listeners are added only when the method that attaches the event to the element is called. So a newly added item won't normally have an event listener attached to it.

Event Propagation: when an event gets triggered on an element and the event is not handled, the event will bubble up to its parents towards the document root until one of its ancestors handles it.


The second parameter of the on() method can be used to filter out which child triggered the event. The event first gets triggered by the parent container element, but the second parameter checks to see if the triggering element matches that selector.


// Syntax
$("selector").on("event", "selector", functionHandler);

// Example
$("div").on("click", "a", function(event) {
    // statements
});

// Another example
$('#content').on('click', $('input[type="button"]'), function(e) {...});

Retrieving id of element in JQuery vs Javascript


// Returns the id in JQuery
var id = $("selector").attr("id");

// Returns the id in Javascript (handler function previously attached to an event listener)
function someFunction(event) {
    var id = event.target.id;
    var id = this.id;   // also returns the id
}

Scroll smoothly to an Element

The href attribute contains the id for the element to scroll to, so it is grabbed and used to determine the offset from the top of the browser window.


$('a[href^="#"]').on('click', function(event) {
    var anchorHandle = $(this.getAttribute('href'));
    if (anchorHandle.length) {
        event.preventDefault();
        $('html, body').stop().animate({
            scrollTop: anchorHandle.offset().top
        }, milliseconds);
    }
});

Cookies

An ID for a user. Could hold browsing preferences. Has constraints like expiration. Can store in a cookie a key-value pair: a name and a value. Can store 5MB of data on local machine. Local storage (persistent)and local sessions (disappears for one session) to minimize connections to server. Cookies also help minimize connections to server. All are sandboxed, can't get to other people's information. Nowadays, there are APIs to get information from other servers.
