HTML

HTML (Hyptertext Markup Language): a markup language to create and add structure to web pages. A document containing text.

  • Syntax uses elements to mark up document.
  • Elements are defined with angle brackets <>.
  • Elements contain attributes which can each be set a value.

Document Object Model(DOM): tree-like structure but not binary, the structure of HTML wherein each element has an object associated with it which can be referenced in JavaScript. Elements can be selected in CSS.


index.html is the first page loaded on the website


Doctype Declaration: tells browser to load most current version of HTML, not a tag.

Forms

<form action="" method="" name="" enctype="">: creates a form. action attribute defines action to be performed when form is submitted. Usually to webpage on server (a php file). method attribute specifies GET or POST HTTP methods. Use GET for search or lookup, submitted form data will be visible. POST when changing data on the server, sends the form data to be processed. name attribute for different forms, must have for each input field or else data won't be sent. enctype for how the submitted data will be encoded.

Form Attributes

value: sets initial value of input

readonly: input is readonly, can't be set

disabled: input field is disabled and can't be submitted

size: size in characters for input field

maxlength: maximum allowed length for input

autocomplete: if form should autocomplete, browser completes input values based on previously entered values

autofocus: input field should get focused when page loads

placeholder: gives hint describing expected value, the lightened text will disappear when a value is inputted by user

pattern: takes a regular expression to check against input's value

required: indicates that field must be filled out, can't be empty, will set it as invalid

There are many other attributes, check out the w3schools website to learn more.

HTML Entities

Some characters are special in HTML and to specify it literally, it must be escaped. In this case, instead of escaping with backslash, we escape with the ampersand &

&nbsp; - non-breaking space

&lt; - less than <

&gt; - greater than >

&amp - ampersand &

&quot - quotation mark "

&apos - apostrophe '
